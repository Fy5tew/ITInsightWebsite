# IT Insight Website
Курсовой проект БГТУ 2 семестр 1 курс
## Запуск
Перейти в директорию проекта

    cd /path/to/project/

Запустить сервер

    http-server -p 8000 --cors
Открыть сайт в браузере

    http://localhost:8000/index.html
